scheme-compiler
===============

(I need to come up with a better name.)

A small, self-contained compiler for a Scheme-like language written in Ruby.

Quick Start
-----------

```sh
git clone https://github.com/avik-das/scheme-compiler.git
cd scheme-compiler
bundle install
./s-expr.rb test.scm
./main
```

Running the tests
-----------------

```sh
./run-tests.sh
```

SDL2 Example
------------

```sh
cd sdl2-test
make
./marley
make clean
```
